## Getting started

### [nvm](https://github.com/creationix/nvm)

- [`/.nvmrc`](./.nvmrc) specifies the node version for this repo.

```bash
# install modules
npm install

# run development
npm run dev

# and in another tab
npm run test:watch
```

## Testing

```bash
npm run test
```

### [eslint](http://eslint.org)

```bash
# will throw linting errors
npm run lint

# will do above and my fix some
npm run lint -- --fix
```

## Serving

```bash
npm run build
npm run serve
```
